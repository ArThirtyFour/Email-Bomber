## Installation

Download the tool:
```bash
$ git clone https://github.com/mohinparamasivam/Email-Bomber.git
$ cd Email-Bomber
```

Run it with python 2 (deprecated): 
```bash
$ python2 emailbomber.py
```

Run it with Python 3: 
```bash
$ python3 emailbomber3.py
```
